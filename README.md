⚡ Electricity Billing System (Java)

A Java-based Electricity Billing System developed using Java Swing, JDBC, and MySQL to automate the process of electricity consumer management, meter details, bill generation, and payment tracking. This project provides an easy-to-use graphical interface for administrators to manage customer records, calculate electricity bills based on meter readings, and maintain billing history efficiently.

📌 Features
🔐 Secure Admin Login
👤 Customer Registration & Management

📊 Automatic Electricity Bill Calculation
📅 Monthly Bill Generation
💳 Payment Management
📜 Bill History & Record Maintenance
🖨️ Printable Bill Details

💡 How It Works
Admin logs into the system.
Register new electricity consumers.
Add and manage meter details.
Enter current meter readings.
System calculates electricity charges automatically based on units consumed.
Generate and save monthly bills.
Record payments and update bill status.
View customer information and billing history.
🎯 Project Objective

The objective of this project is to replace manual electricity billing with a computerized system that improves accuracy, reduces paperwork, minimizes calculation errors, and provides efficient customer and billing management.
